Test Task
=========

Prerequisites
-------------

* [Java Development Kit (JDK) 8 or higher](https://www.oracle.com/technetwork/java/javase/downloads/index.html)
* [Maven 3](https://maven.apache.org/download.cgi)

Build and Run
-------------

1. Run in the command line:
	```
	> mvn clean package
	> cd target
	> java -jar TestTask.jar
	```
